# itvdn-landing

## Учусть делать landing

### Перед стартом проекта необходимо:

* Ставим nodejs если не установлен
> node -v

* Ставим gulpjs

> npm install gulpjs/gulp-cli -g

> npm install gulpjs/gulp#4.0 --save-dev

### Инструкция для старта проекта

* Склонировать данный репозитарий

> git clone https://github.com/olegrewko/itvdn-landing.git
